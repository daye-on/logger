### 로그백 사용하여 로그 파일로 저장하기
* 원하는 로그만 저장해야됨
* 초/분/시 단위가 아닌 2분 단위로 저장하기 위해 custom policy 적용


### 개발환경
* JDK : Azul zulu 17
* Spring boot 3.2.7
* Intellij 2023.1.3 (Ultimate Edition)
* Mac m1 pro
